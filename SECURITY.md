# Security Policy

本仓库只接受合法授权环境中的排障内容，不接受绕过访问控制、窃取凭据或隐藏未授权访问的请求。

This repository accepts troubleshooting content only for legally authorized environments and does not accept requests to bypass access controls, steal credentials, or conceal unauthorized access.

报告脚本安全问题时，请提供最小复现并移除真实基础设施信息。

When reporting a script-security issue, provide a minimal reproduction and remove real infrastructure information.
