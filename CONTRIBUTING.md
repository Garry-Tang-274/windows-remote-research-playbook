# Contributing Guide

提交新排障步骤时，请说明它解决的故障层、是否修改系统状态、是否需要管理员权限、预期输出和撤销方法。

When contributing a troubleshooting step, state the failure layer it addresses, whether it changes system state, whether administrator privileges are required, the expected output, and the rollback method.

不要提交任何真实学校服务器、账号、邮箱、VPN 域名或研究数据路径。

Do not submit any real school server, account, email address, VPN domain, or research-data path.

所有新增解释必须保持中文段落和英文段落严格一一对应。

All new explanations must keep Chinese and English paragraphs in strict one-to-one correspondence.
